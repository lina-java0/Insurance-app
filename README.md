Insurance App — backend-приложение для расчёта страховой премии по туристической страховке.

Архитектура
- Проект построен на основе Hexagonal Architecture (Ports & Adapters).
- Бизнес-логика полностью сосредоточена в core
- REST — inbound адаптер
- RabbitMQ — outbound адаптер

Инфраструктурные зависимости изолированы от доменной логики

Такой подход обеспечивает:
- независимость бизнес-логики от фреймворков
- удобное тестирование
- гибкость при расширении системы

Tech Stack
- Java 21
- Spring Boot 3
- Spring Web
- Spring Data JPA
- RabbitMQ (в разработке)
- Liquibase
- H2 / MySQL
- Gradle
- JUnit 5
- Mockito

База данных
- Настроены Spring Profiles: h2 и mysql
- Используется Liquibase для версионирования и миграций схемы БД
- Возможность запуска как в in-memory режиме, так и с MySQL

Тестирование
- Unit-тестирование бизнес-логики с использованием Mockito
- Интеграционные тесты для проверки взаимодействия слоёв

Дополнительно
- Реализована frontend-часть для работы через веб-интерфейс

Проект находится в активной разработке
